# Intent: organize

Use for converting supplied notes, documents, or ideas into a clearer structure without adding unsupported conclusions.

## Thinking contract

- Preserve the source's meaning, attribution, uncertainty, and important ordering.
- Merge duplicates only when they express the same claim; keep conflicting statements visible.
- Separate source facts, source opinions, user decisions, and AI-proposed grouping.
- Name missing transitions or evidence as gaps instead of filling them with invented content.
- Optimize for retrieval and continued editing, not exhaustive paraphrase.

When reorganizing the current Page, retain every unaffected node and use minimal merge operations.
