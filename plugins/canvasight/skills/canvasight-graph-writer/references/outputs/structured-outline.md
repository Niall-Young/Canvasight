# Output: structured-outline

Use a top-to-bottom hierarchy for articles, definitions, reviews, and material that has a reading order.

- One root establishes purpose or thesis.
- Section nodes state their distinct responsibility; child nodes contain evidence, examples, questions, or details.
- Split a section when its parts can stand as independently readable sections; do not split supporting sentences that only make sense as one argument.
- Avoid identical depth for every branch when the material does not support it.
- Use vertical layout and preserve source order when organizing existing content.
