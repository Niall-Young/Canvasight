# Domain: research

Use for investigating a question, comparing evidence, testing explanations, or informing a decision.

## Required coverage

| Coverage key | Required content |
| --- | --- |
| `research.question` | Precise research question and the decision or context behind it. |
| `research.scope` | Time, population/object, geography or environment, inclusions, and exclusions. |
| `research.factsSources` | Known facts connected to named sources or supplied evidence. |
| `research.dimensions` | Defined concepts and consistent analysis dimensions. |
| `research.hypotheses` | Plausible explanations or hypotheses stated so evidence can challenge them. |
| `research.evidenceCounterevidence` | Supporting evidence, counterevidence, and alternative explanations. |
| `research.uncertainty` | Information gaps, uncertainty, bias, and source-quality limitations. |
| `research.patterns` | Comparable results, differences, correlations, or repeated patterns. |
| `research.conclusions` | Supported conclusions and explicit claims the evidence cannot support. |
| `research.implications` | What the findings change for the user's next decision or investigation. |

Do not flatten conflicting sources into false consensus. Mark inference and uncertainty at the node where they occur.
